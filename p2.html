Part 1: System Unit Assembly

Lecture Duration: 1 Hour

‚ö†Ô∏è Safety Procedures First!

ESD (Electrostatic Discharge) Prevention

  • ‚Ä¢ Always wear an anti-static wrist strap
  • ‚Ä¢ Work on an anti-static mat
  • ‚Ä¢ Touch a grounded metal object before handling
  • ‚Ä¢ Handle components by edges only
  • ‚Ä¢ Keep components in anti-static bags until needed

General Safety Rules

  • ‚Ä¢ Disconnect power before working
  • ‚Ä¢ Keep workspace clean and organized
  • ‚Ä¢ Use proper tools (magnetic screwdriver, etc.)
  • ‚Ä¢ Never force components into slots
  • ‚Ä¢ Document cable connections before disconnecting

üñ•Ô∏è Essential Components

CPU (Processor)

The "brain" of the computer. Handles all calculations and instructions. Modern CPUs: Intel Core i9, AMD Ryzen 9

üí° Handle by edges, align with socket markers

Motherboard

The main circuit board connecting all components. Contains CPU socket, RAM slots, expansion slots, and chipset.

üí° Install standoffs first to prevent short circuits

RAM (Memory)

Temporary storage for active programs. Current standard: DDR5 (4800-8400 MHz). More RAM = better multitasking.

üí° Align notch with slot, push until clips snap

Storage (SSD/HDD)

Permanent data storage. NVMe SSDs (7000+ MB/s) vs SATA SSDs (550 MB/s) vs HDDs (150 MB/s).

üí° NVMe goes in M.2 slot, SATA needs data + power cables

Power Supply (PSU)

Converts AC to DC power. Choose wattage based on components. 80+ Gold efficiency recommended.

üí° Calculate total wattage, add 20% headroom

Case (Chassis)

Houses all components. Consider: form factor (ATX, mATX, ITX), airflow, cable management, front I/O.

üí° Match case size with motherboard form factor

üìù Assembly Steps (Proper Order)

1

Prepare Workspace & Install CPU

Set up anti-static measures. Lift CPU socket lever, align CPU triangle marker with socket marker, gently place CPU (zero force), close lever.

2

Install CPU Cooler

Apply thermal paste (pea-sized dot) if not pre-applied. Mount cooler, connect fan header to CPU_FAN on motherboard.

3

Install RAM

Open slot clips, align RAM notch, insert at slight angle, push down firmly until clips snap. Use slots A2/B2 for dual-channel.

4

Install M.2 SSD (if applicable)

Remove M.2 heatsink, insert SSD at 30¬∞ angle into slot, push down and secure with screw, replace heatsink.

5

Prepare Case & Install Motherboard

Install I/O shield, mount standoffs matching motherboard holes, carefully lower motherboard, secure with screws (don't overtighten).

6

Install PSU

Mount PSU (fan facing down/up based on case design), secure with screws. Route cables before connecting.

7

Connect Power Cables

24-pin ATX to motherboard, 8-pin CPU power (top-left of board), GPU power (if needed), SATA power to drives.

8

Connect Front Panel & Fans

Connect power switch, reset, HDD LED, power LED (check polarity), USB headers, audio header. Connect case fans.

9

First Boot & POST

Connect monitor, keyboard, power. Press power button. Watch for POST (Power-On Self-Test). Enter BIOS to verify components detected.

üîó Network Topologies

Bus Topology

Single cable, all devices connected. Simple but single point of failure.

Star Topology

Central hub/switch. Most common today. Easy troubleshooting.

Ring Topology

Circular connection. Data travels in one direction. Used in token ring.

Mesh Topology

Every device connected to every other. High redundancy, expensive.

Tree/Hybrid Topology

Hierarchical structure. Combination of topologies. Used in large organizations.

üîå Basic Network Devices

Hub

Basic device that broadcasts data to all ports. No intelligence. Obsolete.

Switch

Intelligent device that forwards data only to intended recipient using MAC addresses.

Router

Connects different networks. Uses IP addresses. Manages traffic between networks.

Access Point (AP)

Provides wireless connectivity. Bridges wired and wireless networks.
