Course Description

This course introduces students to the fundamental concepts of IT infrastructure and computer networking. It covers computer hardware components, system unit assembly, data communication principles, network models, and physical network infrastructure. Students will explore IP addressing, subnetting, VLANs, routing, and essential network services such as DHCP and DNS. The course also includes wireless networking, network monitoring, security principles, documentation, and infrastructure evaluation.

Course Learning Outcomes

2

Analyze network components, OSI layers, IP addressing, and subnetting schemes for different organizational needs.

3

Configure basic network services including VLANs, routing, DHCP, DNS, and wireless networks.

4

Apply network security principles, monitoring tools, and evaluate network performance and reliability.

5

Document, integrate, and defend a basic network infrastructure design following industry standards.
