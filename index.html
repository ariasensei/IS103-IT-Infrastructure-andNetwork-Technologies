üìñ Course Description

This course introduces students to the fundamental concepts of IT infrastructure and computer networking. It covers computer hardware components, system unit assembly, data communication principles, network models, and physical network infrastructure. Students will explore IP addressing, subnetting, VLANs, routing, and essential network services such as DHCP and DNS. The course also includes wireless networking, network monitoring, security principles, documentation, and infrastructure evaluation. Emphasis is placed on hands-on configuration, practical skills, and outcome-based learning to prepare students for real-world networking environments and further studies in information systems.

üéØ Course Learning Outcomes

By the end of the semester, students are able to:

  • Explain fundamental concepts of computer hardware, data communication, networking models, and network infrastructure.
  • Analyze network components, OSI layers, IP addressing, and subnetting schemes for different organizational needs.
  • Configure basic network services including VLANs, routing, DHCP, DNS, and wireless networks.
  • Apply network security principles, monitoring tools, and evaluate network performance and reliability.
  • Document, integrate, and defend a basic network infrastructure design following industry standards.
